📋 Overview

Project Overview

The EV3 Driving Base is a two-motor robot controlled through EV3 Classroom's ready-made programs. You'll build it, run the stock programs, observe and record results.

🎯
Learning Objectives
Understand how a two-motor differential drive works. Use EV3 Classroom's stock movement blocks. Observe how speed and distance settings affect robot behaviour.
⚙️
How It Steers
Two large motors — one per wheel — run independently. EV3 Classroom's Move Steering block controls both at once. Steering 0 = straight; negative = left; positive = right.
📦
Stock Programs
EV3 Classroom includes pre-built programs in the Driving Base unit. Use these as your starting point, adjust settings, and document what changes.
📝
Student-Driven
You decide what settings to try. No single correct answer — experiment, measure each result, and write down what you notice.

🔧 Build Guide

Build Steps

Work through each step in order. Take photos after each stage for your documentation.

01
Plan Your Layout
Sketch a top-down view of your robot before touching any pieces. Decide where the brick, motors, and wheels go. The centre of gravity should sit between the two drive wheels.
💡 Keep it compact and low. The sketch counts as part of your documentation.
02
Mount Both Large Motors
Attach the two large servo motors symmetrically on opposite sides of the chassis. Both output shafts must be on the same horizontal axis or the robot will drift when driving straight.
💡 Use a beam across both motors to lock them at the same height. Uneven motors = uncontrollable drift.
03
Attach the Drive Wheels
Press the large traction wheels onto the motor axles and lock them with a bushing. Both wheels must be the same size and fully seated.
💡 Check that both wheels spin freely by hand before moving on.
04
Add the Caster Wheel
Mount a small caster at the front or rear. It must touch the ground evenly and let the robot pivot freely. Use spacers to match the height of the drive wheels.
💡 If the robot tips forward or backward, adjust the caster height first.
05
Mount the EV3 Brick
Secure the EV3 brick so the screen faces up and motor ports B and C are accessible. Centre it between the wheels to keep weight balanced.
💡 The brick is the heaviest part. Off-centre placement causes drift that programming can't fix.
06
Connect Motor Cables
Plug the left motor into Port B and the right motor into Port C. Route cables so they cannot catch in the wheels.
💡 EV3 Classroom stock programs expect B = left, C = right. Swapping makes turns go the wrong way.
07
Manual Roll Test
Push the robot by hand on a flat surface. It should roll straight. If it drifts, fix the mechanical issue now — no program can correct a physical misalignment.
💡 Document anything you adjusted here. It's part of your engineering process.

📋 Overview

Project Overview

The Color Sensor Sorting project uses the EV3 color sensor to detect different colors and trigger matching robot responses using Switch blocks.

🎯
Learning Objectives
Learn how the EV3 color sensor detects reflected light. Use Switch blocks to make decisions based on color. Understand how sensors give robots awareness of their environment.
🎨
How the Color Sensor Works
The sensor shines a light downward and reads what bounces back. It identifies the color of the surface beneath it. Plug it into Port 1 and point it at the colored card you want to scan.
📦
Stock Programs Used
EV3 Classroom includes pre-built color sensor programs in the Color Sensor unit. These use the Switch block to branch based on what color is detected.

🔧 Build Guide

Build Steps

You can use your existing Driving Base or build a simple stationary frame. The key requirement is correct sensor placement.

01
Start With the Driving Base
Your Driving Base from Project 1 is the starting point. You'll add the color sensor to the front of the robot pointing downward at the floor.
💡 If your Driving Base is already working, don't rebuild it — just add the sensor mount.
02
Build the Sensor Mount
Use Technic beams to create a bracket that holds the color sensor at the front of the robot, facing straight down toward the surface. The ideal height is 1–2 cm above the ground.
💡 Too high and the sensor gets confused by surrounding light. Too low and it scrapes the ground.
03
Attach the Color Sensor
Mount the EV3 color sensor onto the bracket so the light window faces directly downward. Secure it firmly — a wobbly sensor gives inconsistent readings.
💡 The sensor has a small LED ring around the lens. When plugged in and active, it will glow the color it detects.
04
Connect the Sensor Cable
Plug the color sensor cable into Port 1 on the EV3 brick. Route the cable so it doesn't interfere with the wheels or get caught on anything.
💡 EV3 Classroom's stock color programs expect the sensor on Port 1 by default.
05
Prepare Your Color Cards
Lay out your colored cards or tiles on a flat white surface. Make sure each card is large enough for the robot to pause over it — at least 10×10 cm.
💡 Use bright, saturated colors. Faded or low-contrast colors may not register correctly.
06
Sensor Test
In EV3 Classroom, open the Port View on the brick display and move the sensor over each colored card. Confirm it correctly identifies all 4 colors before running any programs.
💡 If a color is misread, try adjusting the sensor height or the room lighting.
